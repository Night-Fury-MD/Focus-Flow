# 🌊 FocusFlow - Modern Task Manager

**FocusFlow** is a sleek, minimalist task management dashboard designed to help you maintain your flow and stay organized.

## ✨ Features
- 🌓 **Smart Dark Mode**: Seamless transition between light and dark themes.
- 📊 **Dynamic Progress**: Visual progress bar and real-time task statistics.
- 📂 **Smart Views**: Filter between All, Pending, and Completed tasks.
- 💾 **Local Storage**: Your tasks stay saved even after closing the browser.
- 📱 **Fully Responsive**: Optimized for desktop and mobile devices.

## 🛠️ Built With
- **HTML5** (Semantic structure)
- **CSS3** (Flexbox, Grid, Variables & Animations)
- **JavaScript** (ES6+, DOM Manipulation, LocalStorage API)

## 🚀 How to use
Just open `index.html` in your browser and start flowing!
